View file information

To access the detailed information about the currently edited spreadsheet, click the File icon at the left sidebar and select the Spreadsheet Info option.

General Information

The file information includes spreadsheet title, author, location and creation date.

Note: Online Editors allow you to change the spreadsheet title directly from the editor interface. To do that, select the Rename... option from the File menu or click on the file name displayed in the editor header next to the logo (in the upper left corner), then enter the necessary File name in a new window that opens and click OK.

Permission Information

Note: this option is not available for users with the Read Only permissions.

To find out, who have rights to view or edit the spreadsheet, select the Access Rights... option at the left sidebar.

You can also change currently selected access rights clicking the Change access rights button in the Persons who have rights section.

To close the File pane and return to your spreadsheet, select the Close Menu option.
